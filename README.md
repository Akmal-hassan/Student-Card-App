# Student-Card-App


🎓 Student Card Generator Project :
A responsive and interactive Student Card Generator built using HTML, CSS, and JavaScript. This project allows users to enter student details such as name, class, roll number, and upload a profile image to instantly generate a stylish digital student card. The application features a clean modern UI with gradient designs and responsive layouts for mobile, tablet, and desktop screens.
✨ Features:


Dynamic student card generation


Image upload and preview feature


Responsive design for all devices


Modern gradient UI styling


Real-time data display using JavaScript


Beginner-friendly project for practicing DOM manipulation and file handling


🚀 This project helped me improve my frontend development skills, especially working with forms, image handling, responsive design, and dynamic content rendering using JavaScript
